# Tic Tac Toe Game

Tic Tac Toe Game just for fun. It's text-based (No GUI)

# How to install

No virtual environnement needed since it used basics library. Then to launch it:
```
python3 main.py
```

# How to stop the game

On UNIX OS: 
- Press Ctrl + C
On Windows:
- Buy a Mac/Linux :)
